Module urban_sarca_lib
-----------------------

Urban Green SARCA library includes functions usable for quick estimation of

Module SEBCS_lib
------------------

Module SEBCS_lib is library of SEBCS software (Brom 2020) designed for
geographical and remotely sensed purposes, mainly for analysis of energy
balance of the surface. Some methods are used within Urban Green SARCA software.
